# Week 2: AI 辅助全栈开发

使用 AI 编码工具（Cursor）从零开始扩展一个完整的 Web 应用。

## 适合谁

- 想了解"应用是怎么搭起来的"的人
- 对 Web 开发感兴趣但没经验的人

## 前置条件

- 完成 Week 1（理解基本的提示技术）
- 安装 Cursor 编辑器
- 安装 Ollama 和本地模型

## 预计时间

6-8 小时（首次接触 Web 开发），4-5 小时（有一定经验）

## 学习顺序

按以下顺序学习本周内容：

1. **阅读并完成中文指南** → [assignment-zh.md](assignment-zh.md) — 架构解释 + 环境准备 + 练习任务（完整中文版）
2. **记录学习笔记** → [writeup-zh.md](writeup-zh.md) — 结构化笔记模板
3. **进入下一周** → [Week 3: MCP 服务器](../week3/README.md)

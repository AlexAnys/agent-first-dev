# Week 1: 提示工程（Prompt Engineering）

掌握 6 种核心提示技术，理解如何有效地与大语言模型对话。

## 适合谁

- 想了解 AI 对话背后原理的人
- 准备开始使用 AI 编码工具的人
- 不需要编程经验，但需要基本的计算机操作能力

## 前置条件

- 安装 Ollama（本地运行 AI 模型的工具）
- 基本的文本编辑器使用

## 预计时间

4-6 小时（首次学习），2-3 小时（有 AI 使用经验）

## 学习顺序

按以下顺序学习本周内容：

1. **阅读中文导读** → [assignment-zh.md](assignment-zh.md) — 概念解释、日常类比、学习提示
2. **完成课程练习** → [assignment.md](assignment.md) — 原课程作业（英文）
3. **记录学习笔记** → [writeup-zh.md](writeup-zh.md) — 结构化笔记模板
4. **进入下一周** → [Week 2: AI 辅助全栈开发](../week2/README.md)

# Week 8: 多技术栈应用（Multi-Stack Apps）

用三种不同的技术栈构建同一个应用，体验 Vibe Coding 与系统化开发的区别。

## 适合谁

- 想理解"技术栈选择"意味着什么的人
- 想体验 AI 应用生成平台（bolt.new 等）的人

## 前置条件

- 了解基本的 Web 应用概念（Week 2）
- 对前后端概念有基本认知

## 预计时间

8-12 小时（三个版本），6-8 小时（如果有 Web 开发经验）

## 学习顺序

按以下顺序学习本周内容：

1. **阅读并完成中文指南** → [assignment-zh.md](assignment-zh.md) — 概念回顾 + 环境准备 + 练习任务（完整中文版）
2. **（可选）参考英文原文** → [assignment.md](assignment.md) — 原课程作业，供对照参考
3. **记录学习笔记** → [writeup-zh.md](writeup-zh.md) — 结构化笔记模板
4. **完成!** → [返回模块首页](../README.md) · [返回知识库首页](../../../README.md)

## 为什么放在最后

这周是对前 7 周所有知识的综合运用。你会发现：有了前面的积累（提示技术、安全意识、审查能力），你使用 AI 生成应用的方式和 Week 1 时完全不同。

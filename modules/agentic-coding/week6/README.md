# Week 6: 安全扫描（Security Scanning）

使用 Semgrep 扫描代码漏洞，学习 AI 辅助的安全检查流程。

## 适合谁

- 所有使用 AI 写代码的人（安全是每个人的责任）
- 想了解"AI 生成的代码安全吗？"这个问题的人

## 前置条件

- 了解基本的 Web 应用结构（Week 2）
- 不需要安全领域的专业知识

## 预计时间

4-5 小时
